Windows 10 を使ってみたI tried windows 10

2015年に、Windows 10が発売される予定です。

現在、 Windows 10のプレビュー版である、Technical Preview(テクニカルプレビュー)版が公開されています。ここからダウンロードできます。

実際に使ってみた

実際に Windows 10 Technical Preview版を触ってみました。とは言っても、前々(英語しかな い頃)から触っていたのですが、今回、日本語も使えるようになったということで、触ってみま した。

Windows 10 ではここが変わった！

スタートメニューが復活！

Windows 10では、スタートメニューが復活しました。それも、ただ単純に7のモノが復活した だけではなく、Windows 8/8.1 のModern UIのタイルとの合わせ技です。

私は Windows 8.1のスタート画面にだいぶ慣れちゃって、とても便利に使っているのですが、 中にはWindows 7のままが良かったという意見もあるでしょう。そんな問題を、うまく解決し ているのではないかと思います。あくまで個人の意見ですよ！！

やっとかよ！仮想デスクトップ

仮想デスクトップというモノが実装されました。Windows しか触ったことないよ(∩´﹏`∩)ふええ...という人のために説明しますと、
仮想デスクトップは、好きな数だけデスクトップを追加できる機能です。そんなにいっぱい何 に使うの？って思うかもしれないけど、わりとこれ便利な機能です。そんなに使ってないけど(

例えば、あなたが職場で働いているとしましょう。
デスクトップを2個用意して、一個はWord、Excelなど仕事用、もう一個はゲームやブラウザを 立ち上げます。
上司がいるときは、仕事用のほうで真面目そうに仕事します。上司がいなくなった隙にデスク トップをサッと切り替えて遊びます。上司が来た！危ない！！ってときにすぐにデスクトップ を切り替えて仕事します。と、こんなことができます。別に仕事サボれとは言ってないからね ！！

まあこんなたとえはちょっとアレかもしれないので、他の使い方を考えると、
私の場合、いつも、コーディングをしながら、ブラウザを開いていたりします。
Windows 8.1なので、AeroSnapで、左にVisual Studio、右にChromeを立ち上げて置くん ですが、どうしても複雑なアプリだと、狭すぎて辛いのです。
そこで、仮想デスクトップの出番。デスクトップ1全体にVisual Studio、デスクトップ2に ChromeとかTwitterとかいろいろ開いておけば、どちらもデスクトップを目一杯使って利用す ることができます。

ストアアプリがウィンドウになった！

これもやっとか！っていう機能だと思います。
これまで画面を専有していたストアアプリを、ウィンドウのように好きな所に表示できます。
PC設定など、ストアアプリでしかできないようなことを、ウィンドウの上でできるのはわりと 便利なものです。

また、ユニバーサルアプリという概念があり、ストアアプリとはちょっと違い、左上に三のマ ークがなく、より自由度の高い扱いができるみたいです。新しいストアやマップはこの仕組み でできているみたいです。

検索パネルがタスクバーに登場！

検索するための窓が登場しました。エクスプローラーを開かずに、すぐにファイルを検索でき ます。OS XのSpotlightのようなモノです。

しかし、電卓の計算結果などが表示できればもっといいんだけどなと思いました。そのため、 フィードバックアプリに意見を書きました！

通知センター登場！

これまたOS XやiOSのような、通知センターが、 Action Centerという名前で登場しました！

これまで、Action Centerは、ドライバが入ってないよーとか、ウイルス対策してないよーとか、バックアップとってない よーとかいうヒントを表示してくれる機能でしたが、今度から、通知をためて表示してくれま す。

この機能は、私もかなり欲しいなと思っていました。なぜなら、通知が来た！と思って見よう としたとこで通知の表示が終わってしまって、えぇ...って思うことがかなりあったからです(私 だけかもしれないけど)。

ただし、通知が溜まり過ぎると消すのがつらくなるかもしれません(´･_･`)

コマンドプロンプトがより高機能に！

これまで、コマンドプロンプトの操作感は最悪でした。ウィンドウサイズはあまり変更ができ ないし、コピペも楽にできませんでした。

Windows 10では、柔軟なコピペができるようになり、ウィンドウサイズもかなり自由に変更で きるようになりました！

...のはずなんですが、私の環境ではなぜかこれまでどおりの仕様で、しかもフォントが指定で きないのです┗(⌒)(╬´･_･`╬)(⌒)┛

電卓がユニバーサルアプリになった！

電卓アプリが、ユニバーサルアプリになりました！関数電卓が見当たらないかもですが、指数 モードがまさにそれです。なんで指数なんだろ...

さらに嬉しいところで、プログラマー電卓で、選んだ進数以外の進数での表示も見ることがで きるようになりました！
これまでは、16進数の値を10進数にしたいとき、まず16進数を選んで、値を入れて、その後10 進数を選ばないと見れず、効率が悪かったのですが、

このように、選んでいない進数での表示も見られます。これは捗りそうです！

Windows 10について思ったこと

日本語が気持ち悪い

Windows 10から、新しく Yu Gothic UIというフォントが追加されます。

游ゴシックは、Windows 8.1から搭載された日本語フォントで、わりとまともなフォントだと 個人的に思ってます。Yu Gothic UIはそれを、アプリの狭いスペースで文字を多く表示できる ように改造したもののようです。

しかし、このフォント、ひらがなとカタカナが不自然に見えます。なんででしょう。

なんと、ひらがなとカタカナの幅が狭くなっているのです。しかも、バランスが全く考慮され ずに。

ひらがな、カタカナだけだとまだマシかもしれませんが、漢字が混ざってくるとかなり不自然 です。

アプリを使っているところです。
かなり字の形が不自然で、気持ち悪いです。それだけでなく、バランスも全く考慮されていま せん。

製品化されるまでには、マシなフォントにするか、自分でフォントを選べる仕様になっていて ほしいものです。

事実、多くのInsider Program ユーザーも困惑しており、改善を求める声が多いことが、フィー ドバックアプリで確認できます。

キーボードの挙動がおかしい

今回VMで実験しているので、そのせいなのかどうかは未検証ですが、たまに、日本語を打と うとして、デスクトップが切り替わったり、検索モードになったり、ということがあります。

どうやら、常にCtrlキーが押された状態になることがあるみたいで、その上でショートカット キーを押してしまっているようです。

一旦デスクトップにフォーカスを当ててから、アプリに戻ることで直ることが多いです。

どうなっているんだ(´･_･`)

検索パネルがバカでかい

見ての通り、デフォルトだと検索パネルが場所を取っています。
タスクバーのプロパティで、アイコンのみ表示できるようにすることができますが、それでも 最初、このように表示されたらちょっと不愉快です(´･_･`)

また、これら検索パネルと仮想デスクトップアイコンの位置を変更できたらいいと思います。

アイコンがあまり良いデザインでない

Windows 10では、アイコンもフラットなものになりました。しかし、あまり良いデザインとは 思えません...

アイコンもフラットにするという発想自体は良いのですが、出来がよろしくないと萎えます(´ ･_･`)

こんなデザインになるくらいなら、元のアイコンでいいんじゃないかな、と思います。

アイコンの整列がなんかおかしい

アイコンを等間隔に並べさせようとすると、変な間ができます。
これ、この隙間にアイコン置けるでしょ...

ネットワークの設定のためになんで待たせる

8.1では、ネットワークの設定をするとき、右からにょーんってアレが出てきますが、10だと、 わざわざ設定アプリが起動します。
待ち時間が微妙にかかるのでこれはどうかと思います。
