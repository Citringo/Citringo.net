C言語風のスクリプト言語。ゲーム開発のために作っている。

実装しているもの

  • 算術演算子 比較演算子 ビット演算子 単純代入演算子 複合代入演算子
  • 文字列型 数値型(全部double) 真偽値型
  • 文字列同士の加算、文字列と数値による乗算ができる
現時点では、基本的な演算および変数への代入のみ対応。

メモ:
for (int i = 0; i < 10; i++) { }
if (a <= 4) { }
while (true) { }
do { } while (true);
switch (hoge) { case 0: print("ぜろ"); case 1: print("いち"); }

switch文は、caseによって暗黙的に分岐の終わりを表す(break;の必要をなくす)ようにしたいところ
