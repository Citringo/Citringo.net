サーバーを移管しました(二回目) Server transferred ============================================================ Citringo.net のサーバーを、 **Hostinger** から **ConoHa** へ移管しました。そりゃいかんな ## 理由 8月25日、 hostinger.jp から、 ```` 無料レンタルサーバーの数を2つまでに制限し、無料ユーザーのFTPアクセスを制限する ```` という内容のメールが来ました。 hostinger の名誉のために言っておくと、これには理由があり、 hostinger のサービスを利用した詐欺/迷惑行為が増え続けていて、その対策のために行った、とのことのようです。 しかし、外部からのFTPアクセスができないとなると、現在行っている GitHub からの自動デプロイができなくなってしまうのです。 そのため、別のサーバーを探すことになりました。 ## ConoHa とは？ ConoHa とは、 GMOインターネットグループが提供する VPS ホスティングサービスで、私は一番安い 1GB プランを 900 円/月で利用しています。 今度はレンタルサーバーではなく、VPSサーバーであるため、かなり高度な利用が可能となっております。 そのため、今後はより様々なことに利用していこうかなと思っています。
