# Citringo.net
The WebSite of Citringo.

Citringo サークルのウェブサイトです。

http://citringo.net
このサイトは、ページの部品を、phpで合成して表示します。そのため、どのページでも同じヘッダー、フッターを見ることができます。
ページは、Markdown で記述します。

## Technology using
- HTML5
- CSS3
	- Bootstrap
- Javascript
	- JQuery
- PHP
